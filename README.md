# Image to ICO Converter Microservice

Это микросервис на базе **FastAPI** и библиотеки **Pillow (PIL)** для мгновенного конвертирования изображений (JPEG/PNG/WebP) в формат `.ico` (иконки, фавиконы).

## 🚀 Особенности
*   **API на FastAPI:** Асинхронная обработка запросов, высокая производительность.
*   **Pillow Звуковой движок:** Качественное масштабирование и многоразмерные ICO-файлы (включают разрешения 16x16, 32x32, 48x48 и 256x256 внутри одного файла).
*   **Стильный UI:** Полноценный встроенный веб-интерфейс с Drag-and-Drop зоной, выполненный в темном неоновом стиле.
*   **Интерактивная дока:** Автоматическая генерация Swagger-спецификации по пути `/docs`.

## 🛠️ Запуск микросервиса

Поскольку у вас настроено портативное окружение Node.js/Python, вы можете установить зависимости и запустить сервер:

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запуск сервера:**
   ```bash
   uvicorn main:app --reload --port 8000
   ```

3. **Работа с сервисом:**
   * Откройте в браузере: `http://localhost:8000` (для работы с веб-интерфейсом).
   * Откройте: `http://localhost:8000/docs` (для работы со Swagger API).

## 🐳 Использование API (cURL)
Вы можете отправлять запросы напрямую из других сервисов:
```bash
curl -X POST "http://localhost:8000/convert" \
     -F "file=@your_image.png" \
     -F "size=all" \
     --output icon.ico
```
Допустимые значения для параметра `size`:
*   `all` (по умолчанию) — 16x16, 32x32, 48x48, 256x256.
*   `16` — 16x16.
*   `32` — 32x32.
*   `48` — 48x48.
*   `256` — 256x256.
